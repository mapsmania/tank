TANK OF DUTY

Neon Twin-Stick Tank Combat

Leave empty for single player, or create/join a session

How to Play

  • WASD - Move tank
  • Mouse - Aim turret
  • Left Click - Fire
  • Goal - First to 10 kills wins!

Multiplayer Setup

  • Uses TripGeo TeamHub for signaling
  • One player creates a session code (e.g., "TANK1")
  • Other players join using the same session code
  • Supports 2-8 players per session
